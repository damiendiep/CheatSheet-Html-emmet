Voici quelques syntaxes en HTML

La balise "p" Créer un paragraphe ou un texte

La balise "b" Rendre le texte en gras

La balise "a href" Créer un lien

La balise "i" ou "em" Met le texte en italique

La balise "u" Souligne un texte

La balise "strong" Met le texte en gras et rend le mot important pour les moteurs de recherche

La balise "img src" Pour inserer une image ex : "img src="lien de l'image" alt="text""

La balise "ol" Permet de créer une liste ordonnée

La balise "ul" Permet de créer une liste non ordonnée

La balise "li" A utiliser dans les balises "ol" et "ul" pour créer ces élements

La balise "tr" Permet de créer un tableau

La balise "th" A utiliser dans le "tr" pour créer une entête de tableau

La balise "td" A utiliser dans le "tr" pour remplir une colonne




Voici quelques syntaxes en CSS

Display Sert à déterminer le position des élèments dans le block

Position Sert à déterminer la position d'un élèment

width Sert à définir la longueur de l'élèment

height Sert à définir la hauteur de l'élèment

Color Sert à définir la couleur de l'élèment

Background-color Sert à définir la couleur du fond de l'élèment

Text-align Permet de choisir le positionnement d'un texte

Voici quelques syntaxes sur EMMET

> Permet d'écrire en une seule fois des sous-balises ex : nav>h1 crée un h1 dans une nav

+ Permet d'écrire une balise à la suite ex : div+p crée une div qui se referme suivi d'un p

Les parenthèses Permet de créer une ligne à l'intèrieur d'une autre ex : div>(nav>div)

* Permet de multiplier une balise ex : nav>p*4 crée 4 fois le p

$ Permet de numéroter de façon croissante des classes ex : li.item$*5

raccourci . Permet de créer une classe ex : div.header crée une classe(header) attribué à la div

raccourci # Permet de créer un id ex : div#header crée un id(header) attribué à la div
